# Extractor de Texto de PDF con Interfaz Gráfica

Esta sencilla aplicación en Python te permite extraer texto plano de un archivo PDF a través de una interfaz gráfica fácil de usar.

## Características

- Selecciona un archivo PDF usando el botón "Seleccionar PDF"
- Extrae el texto del archivo PDF seleccionado
- Muestra el texto extraído en una ventana con scroll

## Requisitos

- Python 3.x
- Biblioteca PyPDF2
- Biblioteca tkinter (generalmente viene preinstalada con Python)

## Instalación

1. Clona o descarga este repositorio en tu máquina local.

2. Instala las bibliotecas requeridas usando pip:
   ```
   pip install PyPDF2
   ```

## Uso

1. Ejecuta el script `pdf_text_extractor_gui.py`.
   ```
   python pdf_text_extractor_gui.py
   ```

2. Haz clic en el botón "Seleccionar PDF" para elegir un archivo PDF en tu computadora.

3. Haz clic en "Procesar" para extraer el texto del PDF seleccionado.

4. Una nueva ventana mostrará el texto extraído.

## Notas

- Asegúrate de tener Python instalado en tu sistema.
- El texto extraído puede no ser perfecto, especialmente para diseños de PDF complejos.
